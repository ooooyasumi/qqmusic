# DESIGN.md — 同担默契局

## 产品意象
把"测试"做成一份**艺人杂志的副刊**：深夜打开 QQ 音乐时，那种深色+歌手海报+情绪文案的感觉；又像翻看网易云年度听歌报告的某一页——专辑封面做模糊背景、巨大的衬线数字、克制的呼吸感。**不是问卷，是一张歌单的情书。**

## 气质关键词
- **深夜电台的衬线排版**（不是圆角卡片+科技蓝那套）
- **杂志感的层级**（巨号数字 / 短句标题 / 注释脚注）
- **克制的色温**（深紫蓝主色，艺人色只作为高光出现）
- **氛围光**（多色光斑、毛玻璃、细颗粒、波形）
- **柔顺过渡**（0.5–0.8s ease-out，没有弹跳）

## 视觉策略
- **背景**：深紫蓝径向渐变 `#0d0a1f → #1a1530 → #251b3d`；上方叠一层艺人色光斑（`blur-3xl`），下方一层细颗粒噪点（`feTurbulence`）
- **巨号数字**：默契分数用 220–320px 衬线斜体显示，带轻微光晕
- **分割线**：细如发丝的 0.5px 横线 + 右侧小字脚注
- **节标题**：全大写英文 + 中文章名并置，英文间距 0.3em
- **氛围元素**：动态音频波形（用 SVG 模拟柱状条）、唱片纹理、年份水印（`2026`）

## 配色方案
- `--bg-base: #0d0a1f`（深夜紫蓝底色）
- `--bg-elevated: rgba(255,255,255,0.04)`（卡片玻璃）
- `--ink-primary: #f4ecd8`（米色衬线主文字，象牙白）
- `--ink-secondary: rgba(244,236,216,0.62)`（次要文字）
- `--ink-mute: rgba(244,236,216,0.35)`（脚注、辅助）
- `--accent-gold: #d4af7a`（暖金，分数高光）
- `--accent-line: rgba(244,236,216,0.18)`（分割线）

**艺人色（仅作为按钮/光斑高光，不做大面积填充）：**
- 周杰伦：青苔绿 `#7fb069`
- 林俊杰：深海蓝 `#5b8def`
- Taylor Swift：玫瑰金 `#e8a5b8`
- BLACKPINK：明黄 `#f5c842`

## 字体排版
- **Display（巨号）**：`Playfair Display`（英文斜体）+ `Noto Serif SC`（中文）— 分数、艺人名、章节大标题
- **Title（章节）**：`Cormorant Garamond`（英文） + `Noto Serif SC`（中文）
- **Body（正文）**：`Source Han Serif SC` / `Noto Serif SC`
- **Caption（脚注）**：`Inter`（小号无衬线，弱化存在感）
- **小标签**：`JetBrains Mono`（编号、ID）

**节奏策略**：
- 数字巨大、与正文形成 4–6 倍尺寸差
- 章节标题前后留 96–128px 空白
- 段落最大宽度 640px，居中
- 全文使用 1.6–1.8 行高，宽松呼吸

## 动效与交互
- **页面切换**：透明度 + 8px Y 位移，0.5s ease-out
- **按钮 hover**：背景透明度变化 + 0.3s 缓动，无位移
- **艺人卡片 hover**：光斑变亮 + 卡片轻微上浮 4px
- **选项卡 hover**：边框由内向外展开 0.3s
- **进度条**：丝滑 0.6s ease-in-out
- **大数字进入**：先 scale 0.9 + opacity 0 → 1，0.8s

## 设计禁忌
- ❌ 不要使用 Tailwind 默认的圆角卡片 + 阴影组合
- ❌ 不要用 Space Grotesk、Inter、Roboto 等通用字体做主标题
- ❌ 不要让艺人色填满整张大背景，只做光斑/按钮/重点高光
- ❌ 不要用 emoji 做装饰（仅作为分隔点的 · 可保留）
- ❌ 不要出现"立即参与"等模板 CTA 文案，统一改成"开始作答" / "发给好友"
- ❌ 不要给艺人卡片加厚厚的白色边框
- ❌ 禁止使用科技蓝渐变、紫蓝渐变作为整体背景（用深紫蓝而非冷蓝）
